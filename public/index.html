Get Started in 2 Minutes

1

Get API Key

Visit groq.com for free key

2

Add Workflow

Create .github/workflows/buildlens.yml

3

Add Secret

Add GROQ_API_KEY to GitHub secrets

4

Done!

Open a PR, BuildLens verifies automatically

Workflow File (.github/workflows/buildlens.yml)
name: BuildLens PR Verification
on:
  pull_request:
    types: [opened, synchronize]

jobs:
  buildlens:
    runs-on: ubuntu-latest
    steps:
      - uses: Jnyanu18/buildlens-ai@v1
        with:
          groq-api-key: ${{ secrets.GROQ_API_KEY }}
          github-token: ${{ secrets.GITHUB_TOKEN }}
